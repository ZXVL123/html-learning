🛠️ 准备工作

开始之前，你需要准备以下三样东西：

1. Git Bash

Windows 上推荐用 Git Bash 执行 git 命令。安装 Git 时会附带。

2. GitHub 账号

如果没有，去 github.com/signup 注册一个。

3. 一个 HTML 文件

比如你已经写好的 html-tags-nesting.html，这是我们要部署的页面。

🎯 下一步

恭喜！你已经掌握了把 HTML 页面部署到 GitHub Pages 的完整流程。

现在你可以：

  • 修改页面内容，然后 git add .、git commit、git push 更新网站
  • 用同样的方法部署更多页面或项目
  • 学习自定义域名，把 username.github.io 换成自己的域名
  • 探索 Jekyll、Hugo 等静态网站生成器，搭建更复杂的站点
